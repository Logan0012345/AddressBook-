# AddressBook-
=> Create Contact is used to add a new contact with name, phone number, and email.
=> Validation is done for every name,phone number and email.
=> List Contactsis used to display all saved contacts, sorting is used by name, phone and email.
=> Search function to search by name, phone, or email.
=> Edit the Contact name, phone, or email of an existing contact.
=> Delete contact from the address book.
=> Checks for duplicate or invalid name, phone, and email.
=> Save and load all the contacts to the file.
=> Data  will be saved in the csv file.
